# Prometheus Energetika Webpage

Landing page for **Prometheus Energetika** built with plain HTML, CSS, and JavaScript.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome (CDN)
- Google Fonts (CDN)

## Project Structure

- `index.html` – page structure and content
- `styles.css` – layout, theme, animations, responsive styles
- `script.js` – interactions and UI behavior
- `assets/` – logo, favicon, and visual assets

## Run Locally

No build step is required.

### Option 1: Open directly

Open `index.html` in your browser.

### Option 2: Use a local server (recommended)

If you have VS Code Live Server:

1. Right-click `index.html`
2. Select **Open with Live Server**

Or with Python:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Editing Guide

- Update section content in `index.html`
- Adjust styling and spacing in `styles.css`
- Modify interactions (scroll, animations, behavior) in `script.js`

## Notes

- This project uses CDN-hosted fonts/icons, so internet access is required for those resources.
